Supported operators:
  • ( and ) :: Isolate a smaller equation inside a bigger one.
  • ^ or e :: Exponentiation
  • * or x :: Multiplication
  • / :: Division
  • % :: Modulo
  • + :: Addition
  • - :: Subtraction
Examples:
  • 1 + 2 * 3
  • 3 + 4 / 2 - 4
  • 4 / 2 ^ 2 - 4
  • 1 + 2 - 3 * 3 + 4 ^ 2
  • 1 - 2 * 3 * 4
  • 7 - 1 * 0 + 3 / 3
